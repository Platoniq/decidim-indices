# Decidim::SurveySections

This is component is a group of surveys.

## Usage

SurveySections will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-survey_sections"
```

And then execute:

```bash
bundle
```
